Inward Paths/Trigger Words:
-Dashboard's catalogue button
-Internal Search
-Filtered Search
-Ascending Sorted Search
-Descending Sorted Search
-Client Profile Page

Outward paths/Call to Action:
-Product View (PRODUCT_VIEW)
-Provides the user a list of all the available products of the company so he can add them to an order

-Search products by name
-Filter products by category and/or specification
-Sort products by price,stock or name, ascending or descending
-Add product to order
-Go to product's page

User/Business Goals:
-Allow the user to add items to the cart for the client's order
-Cart button that allows the user to go the order's checkout page
